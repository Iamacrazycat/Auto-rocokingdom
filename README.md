# Auto-Roco 洛克王国自动化助手

基于https://github.com/yorusacri/Auto-rocokingdom 的改版，主要是重构了一下代码结构，并加入了云打包功能便于分享。因为结构改动有点大不好合并所以没有fork。

## 📅 更新日志
### 2026/4/20
*   **架构重构**：重构为**观察者模式**，实现检测逻辑与策略执行的彻底解耦，系统更稳定且易于扩展。
*   **智能匹配升级**：放弃颜色空间算法。升级挂机决策逻辑，通过 **SIFT 特征匹配** 结合 **HP 血条颜色分析**，精准区分有效战斗（粉色）与野怪遭遇（绿色）。
*   **开发文档更新**：新增 [DEVELOPMENT.md](DEVELOPMENT.md) 开发手册，详细说明项目架构、事件流及二次开发指南。

## 🚀 核心功能
*   **聚能模式 (Mode 1)**：在进入战斗时，自动模拟键盘点击 `X` 键。
*   **逃跑模式 (Mode 2)**：在进入战斗时，自动按 `ESC` 键并点击“是”按钮。
    *   统一使用**物理鼠标点击**。
    *   执行逃跑点击时，请确保“是”按钮露出，不能被其他窗口遮挡。
*   **后台监控**：支持游戏窗口在被遮挡时依然进行图像识别（只要窗口不最小化）。

## ⭐ 建议用法

1. 好友**刷异色**时，上一个属性抵抗目标精灵攻击的精灵（防止暴毙），使用聚能模式，保持窗口非最小化挂机即可。期间其他窗口可以任意覆盖游戏窗口，不妨碍使用设备干其他事。

2. 好友**跑图刷花刷矿**时，可以开逃跑模式，也可以开聚能模式，但需要和好友协商好。

## 📖 使用步骤 (推荐)

1.  **下载程序**：前往项目的 [Actions](https://github.com/yorusacri/Auto-rocokingdom/actions) 或 [Releases](https://github.com/yorusacri/Auto-rocokingdom/releases) 下载最新的打包程序。
2.  **启动运行**：右键点击 `.exe` 文件，选择 **“以管理员身份运行”**（必须，否则无法发送按键）。
3.  **模式选择**：根据命令提示输入 `1`、`2` 或 `3`。

---

## 💡 注意事项
*   **分辨率无关**：得益于 **SIFT 特征匹配** 算法，本项目支持在任意分辨率下运行，无需手动适配或缩放截图。
*   **停止方式**：在任何时候按下键盘上的 **`F8`** 键即可立即安全退出。
*   **窗口状态**：游戏窗口可以被其他窗口覆盖，但**不能最小化**到任务栏，否则图像识别将失效。
*   **逃跑模式点击要求**：逃跑模式使用物理点击，请确保点击瞬间“是”按钮可见且未被遮挡。
*   **审计日志**：每次触发动作都会写入 `logs/audit.log`（JSON 格式），用于排查误触发。
